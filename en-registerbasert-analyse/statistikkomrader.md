# Statistikkområder

<mark style="color:$danger;">En overordnet beskrivelse av emner (statistikkområder) og tilhørende registre, hvilke populasjoner som dekkes og opplysninger som finnes. En mulig inndeling er:</mark>&#x20;

&#x20;

**Demografi**

Registerdata gir oversikt over hvem som bor i Norge: alder, kjønn, bosted, familiesituasjon og sivilstatus. Slike opplysninger er viktige for å forstå befolkningsutvikling, flyttemønstre, fødselstall og aldring i samfunnet.



**Inntekt og offentlige ytelser**

Gjennom skattemeldingsdata og trygderegister får vi detaljert informasjon om folks inntekter, skatteforhold og hvilke ytelser de mottar, som barnetrygd, uføretrygd eller pensjon. Dette gjør det mulig å analysere økonomisk ulikhet og effekter av velferdsordninger.

&#x20;

**Yrkesdeltakelse**

Arbeidsmarkedsdata viser hvem som er i jobb, hvilken type arbeid de har, om de er heltid eller deltid, og hvilke næringer og yrker de jobber i. Dette gir innsikt i sysselsetting, arbeidsledighet og kjønnsforskjeller i arbeidslivet.

&#x20;

**Utdanning**

Utdanningsregistre inneholder informasjon om fullført skolegang, karakterer, frafall og overgang til høyere utdanning. Disse dataene er viktige for å følge utdanningsløp over tid og se sammenhenger mellom utdanning og arbeidsmarked.

&#x20;

**Tjenester fra Nav**

Registerdata fra Nav viser hvilke tiltak og programmer folk har deltatt i, som arbeidstrening, kvalifiseringsprogram eller sykefraværsoppfølging. Dette gir kunnskap om hvordan Navs tjenester fungerer og hvem som bruker dem.

&#x20;

**Helse**

Helsedata fra for eksempel Norsk pasientregister eller fastlegeregisteret gir informasjon om kontakter med helsetjenesten, diagnoser og behandlingsforløp. Selv om tilgangen er strengt regulert, kan slike data gi innsikt i folkehelsen og sammenhengen mellom helse og arbeid, utdanning eller økonomi.
