# En innføring i registerdata

Startside. Sier kort noe om boka, hvem som står bak, pluss lenker til andre relevante ting.

Registerdata som grunnmur for samfunnsforskning<br>

{% tabs %}
{% tab title="Kort oppsummert" %}
* Registerdata har fått en nøkkelrolle i norsk samfunnskunnskap fordi de dekker hele befolkningen, oppdateres jevnlig og gjør det mulig å følge personer og grupper over tid.
* De gir innsikt i sentrale samfunnsområder som utdanning, arbeid, helse, inntekt og bruk av velferdstjenester, og gjør det mulig å studere sosiale forskjeller og livsløp med stor detaljrikdom.
* Denne modulen gir en samlet og praktisk innføring i hvordan registerdata fungerer, og hvordan de kan brukes til å besvare samfunnsvitenskapelige forskningsspørsmål.&#x20;
* Du får lære både om mulighetene og begrensningene, hvordan dataene blir til, hva som påvirker kvaliteten, og hvordan du går frem for å få tilgang til data og bruke dem på en sikker måte.
